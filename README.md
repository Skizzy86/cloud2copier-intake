Single-file HTML — open `index.html`, search for the field name, edit. Common patterns:

```html
<!-- Single field -->
<div class="field">
  <label>Label text <span class="req">*</span></label>
  <input type="text" name="fieldName" placeholder="..." required>
</div>

<!-- Two-column row -->
<div class="field-row">
  <div class="field">...</div>
  <div class="field">...</div>
</div>

<!-- Section -->
<div class="form-section">
  <div class="section-head">...</div>
  ...fields...
</div>
```

## Maintenance notes

- **Formspree first-submission verification**: the very first form submission triggers a confirmation email; you must click the verify link before subsequent submissions are forwarded.
- **Free Formspree tier limits**: 50 submissions/month, plain-text email delivery, no file uploads.
- **Domain & host credentials**: registrar logins from Section 03 may land in your inbox if a client opts to share via the form. Move them to a password manager and delete the email promptly.
- **10-day clock starts on access + content received** — flag any blockers early so the build window doesn't slip.
